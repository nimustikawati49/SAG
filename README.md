# 📘 Sistem Jurnal Pembelajaran & Absensi Digital Guru

## 🚀 Deskripsi

Sistem Jurnal Pembelajaran & Absensi Digital Guru adalah aplikasi web untuk membantu guru mengelola jurnal harian, absensi siswa, rekap otomatis, dan export laporan dalam satu platform terintegrasi, ringan, real-time, serta mendukung digitalisasi administrasi sekolah.

---

## ✨ Fitur Utama

* 📝 Jurnal pembelajaran harian
* 👨‍🎓 Absensi siswa (Hadir, Izin, Sakit, Alpha)
* 📊 Rekap absensi otomatis
* 📁 Riwayat jurnal (edit & hapus terbatas)
* 📤 Export data ke Excel
* 👥 Multi-user (Guru & Admin)
* 🧩 Manajemen data siswa
* ⚡ Validasi data otomatis

---

## 🛠️ Teknologi

* Google Apps Script (Backend)
* Google Spreadsheet (Database)
* HTML, CSS, JavaScript (Frontend)
