# SAG
Sistem Jurnal Pembelajaran &amp; Absensi Digital Guru adalah aplikasi web untuk membantu guru mengelola jurnal harian, absensi siswa, rekap otomatis, dan export laporan dalam satu platform terintegrasi, ringan, real-time, mendukung multi-user, manajemen siswa, serta validasi data untuk mendukung digitalisasi administrasi sekolah.
